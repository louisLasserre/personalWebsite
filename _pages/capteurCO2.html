Création d'un serveur web pour afficher les données

Certainement le projet où je suis le plus fière de ce que j'ai réussi à faire.

En effet, je suis parti d'aucune connaissance, mais un objectif: créer un serveur web en python. J'ai appris, par moi même à utiliser la bibliothèque python flask. Ce projet m'a permis de consolider mes connaissances en python, html et css et de découvrir en pratique l'utilisation des bases de données ainsi que javascript.

Nous avons travaillé à trois personnes pour la réalisation de ce projet.En pleine crise du covid-19 notre professeur nous a demandé de réaliser un système comportant un capteur de CO2 qui devait signalé si il faut aérer la salle de classe ou non pour limiter la propagation du virus. Les données recueillis doivent ensuite être stockées dans une base de donnée puis afficher sur une page web.
